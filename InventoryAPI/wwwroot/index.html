To build and run API in docker

docker build -t api .
docker run -it -p 57877:80 --name mycontainer api
#########################################################

To build and run React app

npm start
#########################################################


Note:

Please check your docker machine ip:

docker-machine ip


#########################################################

I am using docker toolbox in windows. So, my docker-machine ip is http://192.168.99.100. So, from UI, If i want to hit the API, I need to use http://192.168.99.100:57877.

In the UI project, I have set the url as http://localhost:57877/ So, chances are high, if you are using docker in linux machine, it should work directly.

#########################################################

The api swagger doc will be available at http://localhost:57877/swagger or if you are using docker toolbox in windows like me, the url would be most probably- http://192.168.99.100:57877/swagger
